Tutorial

Game:

+ You are trying to color the tiles based on the rules in the rows/columns

+ Ex: if the collumn rule states "4" then that means that for that collumn there will be 4 black tiles with no space in between

+ Ex: if the row rule states "2 2" then that means that for that row there will be 2 black tiles and another 2 black tiles but there is a space in between the black squares

Interatables:

+ Tiles: Click on tiles to flip and change color

+ Give up: Click show solution

+ Reset: Click to reset all tiles to gray

+ Hint: Select a tile after pressing hint button to see solution for that tile

+ Check: Click to check answer
